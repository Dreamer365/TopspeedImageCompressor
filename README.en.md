<h2 id="#feature">特点</h2>

- 超快的压缩速度
- 支持压缩 `GB` 级别的超大图片
- 支持 `7` 种图片格式
- 支持图片预览
- 支持通过拖拽、点击、`ctrl + v` 等多种方式添加图片
- 支持添加文件夹
- 实时显示当前的压缩情况
- 完全在本地执行压缩，不会上传图片，保证隐私安全
